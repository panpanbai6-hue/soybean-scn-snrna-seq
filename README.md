# soybean-scn-snrna-seq
Single-nucleus transcriptome analyses uncover the developmental origins of cyst nematode-induced syncytia and mechanisms of resistance in soybean

## Analysis workflow

1. Quality control and filtering
2. Data integration and clustering
3. Cell type annotation
4. Pseudobulk differential expression analysis
5. Data export and identification of the cell-type specific genes
6. Procambium-to-syncytium trajectory analysis
7. Syncytium developmental trajectory analysis
8. hdWGCNA analysis of F3 syncytium nuclei
